## B. 前台的total顯示格式
增加以下代碼做提取準備。同時判別如果沒SESSION紀錄，先update更新+1並更新此值。(透過我們設計的num+1，函式會自動加1，不用管原本是多少)

### 增添sql.php
```php
//t7
if (empty($_SESSION['visit'])) {
  $_SESSION['visit']="check";
  $post['num+1']=1;
  update($post, "q1t7_total");
} //先判斷是否更新，之後才撈取
$re=select("q1t7_total", 1);
$t7_text=$re[0]['num'];
```
